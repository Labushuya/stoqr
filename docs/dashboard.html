Deployment

▶ Deployment-Guide — Quick Reference ▼
  • Deploy
    docker compose pull && docker compose up -d --force-recreate stoqr
  • Logs
    docker compose logs -f stoqr
  • DB-Migration
    Automatisch beim Start — keine manuelle Aktion erforderlich.
  • Seed (Postgres)
    docker compose cp ./docs/seed.sql postgres:/tmp/seed.sql docker compose exec postgres psql -U stoqr -d stoqr -f /tmp/seed.sql
  • Ersten User anlegen
    curl -X POST https://stoqr.fam.ily/api/auth/sign-up/email \ -H "Content-Type: application/json" \ -d '{"email":"you@example.com","password":"sicher","name":"Admin"}'
  • Weiteren User einladen
    /einstellungen/mitglieder → Einladungslink erstellen
